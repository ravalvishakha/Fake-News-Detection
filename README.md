# Fake News Detection System

This project is a Machine Learning based web application that detects whether a news article is Real or Fake.

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Joblib

## How to Run

1. Install required libraries:
   python -m pip install flask pandas numpy scikit-learn joblib

2. Run the project:
   python fake_news.py

3. Open browser:
   http://127.0.0.1:5000/

## Features
- User can input news text
- System analyzes text using ML model
- Predicts Real or Fake with confidence score
